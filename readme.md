# yacc_float_calculator

## About 

This is a compact calculator with floating point
- calculate real number
- ‘+’ ‘-’ ‘*’ ‘/’ operation
- ‘(’ ‘)’ may appear on expression
- Output to the first digit after the decimal point

Input
-----
2.0*2.5  
5.0*(3.5+2)  
5.25*3  

Output
------
5.0  
27.5  
15.8  

## how to run 

```script
$ make
$ ./calc
```
